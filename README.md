## 🚀 **LIGHTNING-FAST SETUP** ⚡

### 🎯 **Step 1: Clone the FUTURE**
```bash
git clone https://github.com/yourusername/github-traffic-analytics.git
cd github-traffic-analytics
```

### 🌟 **Step 2: Install SUPERPOWERS**
```bash
npm install
# or for the BRAVE
yarn install
# or for the REBELS
pnpm install
```

### 🔑 **Step 3: Configure SECRET POWERS**

1. **Create your GitHub OAuth App** 🏛️
   ```
   🏠 Homepage URL: http://localhost:3000
   🔄 Callback URL: http://localhost:3000/api/auth/callback/github
   ```

2. **Set up your MAGIC variables** ✨
   ```bash
   cp .env.example .env.local
   ```
   
   ```env
   # 🔮 Your SECRET environment spells
   NEXTAUTH_URL=http://localhost:3000
   NEXTAUTH_SECRET=your-ultra-top-secret-code
   GITHUB_CLIENT_ID=your-github-client-id
   GITHUB_CLIENT_SECRET=your-github-client-secret
   ```

### 🎆 **Step 4: LAUNCH TO INFINITY**
```bash
npm run dev
```

**🎉 BOOM! Your app launches at** `http://localhost:3000` **🎉**
